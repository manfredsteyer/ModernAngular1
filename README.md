# ModernAngular1
Angular 1 mit ES6, TypeScript, ES7-Dekoratoren und Component Router (Preview)

## Einrichten
Jedes Verzeichnis stellt ein eigenes Projekt dar. Laden Sie die benötigen Libraries pro Projekt via NPM und JSPM:
```
npm install
jspm install
```
